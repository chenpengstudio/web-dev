Home | Intro | Cheatsheet | Publish (for Adventurers)

GOAL

You will choose a word and recreate a one-page Wikipedia local webpage using html and css code.
Include at least 3 images (can be whatever) and at least 5 paragraphs (copied from Wiki).
Include multiple font styles and colors.
Add an active link to the orignal Wikipedia page.
Bonus points: Create CSS animation and/or add a second page!

WORKFLOW

Step 1: Create a Folder
Create a folder. Name your folder "Your Name Coding Practice".

!!IMPORTANT!!
Add all the images you will be using in this folder.
Your CSS file should be saved in this folder as well.
You will submit your coding homework by uploading the entire folder.

Step 2: Open Folder in Visual Studio Code
Using Chrome browser, go to https://vscode.dev.
On the top-left corner, click "Open Folder" and choose the empty folder you just created.
Click "Select".

Step 3: Create two new files for HTML and CSS code
Click on this icon "new file..."

Name your new file "index.html".
Click on the icon again to create another new file.
Name this new file "styles.css".

Step 4: Declare your HTML file
Click and open "index.html" file
In the first line of the VSCode editor, type in "!", then press your "Enter/Return" key on keyboard.
Your HTML file is now declared.

Step 5: Link your CSS file
Inside of the <head> tag, type in <link rel="stylesheet" href="styles.css">
Now you have linked your CSS file to the HTML file

Start to edit your HTML code inside the <body> tag to write the contents.
Refer to your Cheatsheet for tags.

Step 6: Visualize your webpage
Go to the folder you just created on Desktop.
Open the file "index.html". You will see the local webpage you just created.
Refresh the page to visualize new edits.
