Takeaway

This page include some commonly used HTML tags and CSS rules that you will find helpful.

HTML Tags

Tags Used in <body> </body>

  • First-level heading: <h1>Chapter 1</h1>
  • Second-level heading: <h2>Section 1</h2>
  • Start a paragraph: <p>paragraph</p>
  • Bold text: <b>bold text</b>
  • Italic text: <i>italic text</i>
  • Style a part of a text: <span style="color:#5a5ad0;font-size:larger;">text</span>
  • Create a link to Google: <a href="https://www.google.com">Google</a>
  • Insert a : <img src="https://www.w3schools.com/images/lamp.jpg" alt="Lamp" width="32" height="32">
  • Insert a line break (browers ignore line breaks in HTML code)1: <br>

Tags Used in <head> </head>

  • Set title of a docuemnt: <title>A Meaningful Title</title>
  • Include CSS style sheet style.css: <link rel="stylesheet" href="styles.css">

CSS Selector

Selector Meaning
p {} all <p> elements
h1 {} all <h1> elements
.fancy {} elements of class="fancy"

CSS Property and Value

  • text color: color: red
  • background color: background-color: cyan
  • background gradient 2: background: linear-gradient(90deg, rgba(2,0,36,1) 0%, rgba(0,212,255,1) 100%);
  • font size: font-size: 1.5em
  • font family: font-family: Courier

Footnotes

  1. Browers also ignore extra (more than 1) spaces in HTML code, you need to type &nbsp; to insert additional spaces.↩︎

  2. You can use https://cssgradient.io to help create a background gradient.↩︎
