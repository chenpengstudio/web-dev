Takeaway

This page include some commonly used HTML tags and CSS rules that you will find helpful.

HTML Tags

The invisible setups go between <head> </head>

<head>
Set title of the HTML docuemnt <title>A Meaningful Title</title>
Link the external CSS style sheet style.css <link rel="stylesheet" href="styles.css">
</head>

Visible contents go between <body> </body>

    <body>

Tags for headings

Heading 1 <h1>Chapter 1: HTML/CSS Cheat Sheet</h1>
Heading 2 <h2>Section 1: HTML Tags</h2>

There are 6 levels of heading <h1> through <h6>; <h1> is biggest/first-level heading.

Tags for texts

Start a new paragraph <p>Shepseskaf was a pharaoh of ancient Egypt.</p>
Bold text <b>This sentence will be in bold font.</b>
Italic text <i>An italic sentence.</i>
Insert a line break 1 <br>
Color part of your text <span style="color:#ca5a80;">colored text</span>

Tags to insert links/images

A link to Google <a href="https://www.google.com">Google</a>
An image <img src="images/lamp.jpg" width=32px>
Another image <img src="images/lamp.jpg" width=10%>

You can specify the size of an image in either absolute unit (px for pixels) or relative unit (%).

    </body>

CSS Style Sheets

Recall that there are two ways to specify styles to your HTML documents.

  1. You can declare external CSS inside <head> </head> of an HTML document:

    <head>
  <link rel="stylesheet" href="styles.css">
</head>

  2. You can also use inline style sheet directly in the element’s opening tag:

    <p style="color:red;">A red paragraph.</p>

CSS Selector

We use CSS selectors in an external CSS style sheet to style an entire type/class of HTML elements. In CSS code, we need curly brackets { } to enclose the styles for HTML elements targeted by the selector. Properties of the same element are separated by semicolons ;.

For example, this will give you blue level-one headings in 12-pixel font size:

h1 {
  color: blue; 
  font-size: 12px;
}
Style Selector Correspondance in HTML Contents
p {} all <p> (paragraph) elements in the HTML document
h1 {} all <h1> (heading 1) elements in the HTML document
.fancy {} elements of class="fancy" in the HTML document

CSS Property and Value

CSS for text

text color color: red
font size font-size: 1.5em
font family font-family: Courier
text shadow text-shadow: 1px 1px 2px pink
line height line-height: 3.5rem

CSS for background

background color

background-color: cyan

background gradient 2

background: linear-gradient(90deg, rgb(2,0,6) 0%, rgb(0,212,255) 100%);

background image

background-image: url(images/lamp.jpg);

CSS to center all contents in the HTML body

body{
    max-width: 1000px;
    margin-left:  auto;
    margin-right: auto;
}

For text/image alignment, check out the W3School tutorial.
