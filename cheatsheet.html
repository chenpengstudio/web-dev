Takeaway

This page include some commonly used HTML tags and CSS rules that you will find helpful.

HTML Tags

Body tags
Tag Meaning/Usage
<h1>Chapter 1</h1> first-level heading
<h2>Section 1</h2> second-level heading
<p>paragraph</p> start a paragraph
<b>bold text</b> bold text
<i>italic text</i> italic text
<span style="color:#5a5ad0;font-size:larger;">text</span> to style a part of a text
<a href="https://www.google.com">Google</a> a link to Google
<img src="https://www.w3schools.com/images/lamp.jpg" alt="Lamp" width="32" height="32">
<br> line break
Head tags
Tag Meaning/Usage
<title>A Meaningful Title</title> the title of a docuemnt
<link rel="stylesheet" href="styles.css"> include style sheet style.css

CSS Selector

Selector Meaning
p {} all <p> elements
h1 {} all <h1> elements
.fancy {} elements of class="fancy"

CSS Property and Value

Selector Meaning
color: red text color
background-color: cyan background color
font-size: 1.5em font size
font-family: Courier font family
