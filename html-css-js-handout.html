HTML

<!DOCTYPE html>
<html>
    <head>
        <title>A Webpage</title>
    </head>
    <body>
        <h1>An Interesting Page</h1>
        <p>An exciting paragraph of text.</p>
    </body>
</html>

  • HTML documents have file extension .html

  • An HTML document begins with <!DOCTYPE html>

  • An HTML document consists a series of elements with contents enclosed by <tag>â€™s

  • At a high level,

    • <html> </html> contains everything of the page
    • <head> </head> contains information such as the title and style sheets. These are usually invisible.
    • <body> </body> contains the contents (text, links, images) we see and interact

  • Always complete an HTML element with an opening tag and a closing tag: <p> </p>

  • We will go over some useful HTML tags. For a comprehensive list, check out W3Schools.

CSS

h1 {color: blue; font-size: 12px;}

  • CSS files usually have extension .css

  • A CSS file consists of a list of rules about how to style a HTML document.

  • A selector looks for tags in the HTML to apply the CSS rules

  • Enclose declarations by curly brackets { } and separate them by semicolons ;

  • In an HTML document, you can place CSS inside <head> </head>:

    <head>
  <link rel="stylesheet" href="styles.css">
</head>

HTML Tags

Body tags
Tag Meaning/Usage
<h1>Chapter 1</h1> first-level heading
<h2>Section 1</h2> second-level heading
<p>paragraph</p> start a paragraph
<b>bold text</b> bold text
<i>italic text</i> italic text
<span style="color:#5a5ad0;font-size:larger;">text</span> to style a part of a text
<a href="https://www.google.com">Google</a> a link to Google
<img src="https://www.w3schools.com/images/lamp.jpg" alt="Lamp" width="32" height="32">
<br> a line break
Head tags
Tag Meaning/Usage
<title> title of the docuemnt
<link rel="stylesheet" href="styles.css"> include external style sheets

CSS Selector

Selector Meaning

e.g., .fancy {} elements of class="fancy"

Selector Meaning
